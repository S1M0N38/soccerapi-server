# soccerapi-server

## Installation 

Node and npm are required

1. clone this repo
2. `cd soccerapi-server`
3. `npm install`

## Usage 

Start up the server and leave it running in the background with

```bash
node index.js
```

Now you can make http requests to `http://localhost:5000/` and get a valid `X-Net-Sync-Term`
